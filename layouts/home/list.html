Friendly Scientific Data Analysis

Based on Orange, Quasar empowers researchers from a variety of fields to gain better insight to their data with interactive data visualization, powerful machine learning methods and combining different datasets through easy to understand visual workflows.

Draw Your Workflow

Freely combine components into a workflow that solves your problem. Combine any components that share input and output types. There are infinite possibilities!

For Scientific Data

Quasar add specific file readers, processing, and additional visualizations to a set of powerful Orange components. Open spectral and hyperspectral data sets.

Based on Orange

Orange is a general purpose data analysis toolkit. It is also modular: it can be extended with component packages called add-ons. Quasar is a distribution of Orange with some add-ons preinstalled.

Joint Effort

We made Quasar so that the entire scientific community could freely use it. Quasar is a collaborative effort of many institutions. If you would like to contribute, contact us.
